# JWN's So many dragon eggs Mod

## 개요

엔더드래곤을 잡으면 드래곤 알을 얻습니다.
(<a href="https://github.com/skwodnjs/jwns-mod-so-many-dragon-eggs" target="_blank">github</a>)

## 상세

### 드래곤 알

드래곤 알은 기본적으로 첫번째 드래곤을 잡았을 때에만 엔더의 출구 차원문 상단에 생성됩니다. 
이 모드는 더 이상 이러한 방식으로 드래곤 알을 생성하지 않습니다.
대신, 드래곤을 잡을 때마다 '마지막으로 드래곤에게 공격을 가한 플레이어'에게 드래곤 알을 지급합니다.
