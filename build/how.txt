save 에서 upload 로 옮기고 upload.py 실행하면 업로드 진행

upload.py
1. upload 폴더에 있는 각 md 파일마다
2. 카테고리 유효성 검사
3. id 부여하고 post 폴더로 파일 이동
4. posts.json 폴더에 정보 입력
5. 카테고리의 count 추가

category 의 구조가 변경되거나 post 폴더의 파일이 삭제되면 reset.py 실행

reset.py
1. posts.json 폴더 초기화
2. post 폴더에 있는 모든 md 폴더마다
3. 카테고리 유효성 검사
4. posts.json 폴더에 정보 입력
5. 해당하는 카테고리의 count 추가
